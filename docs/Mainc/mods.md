# 客户端修改限制
::: tip 信息  
* 绿色, 此类Mod完全没有作弊功能的代码  
* 蓝色, 此类Mod有部分代码/缺陷可能导致作弊, 但导致作弊的部分通过反作弊禁用  
* 黄色, 此类Mod有办法代码/缺陷可能导致作弊  
* 红色, 此类Mod是明确禁用的作弊Mod  
目前仅支持说明Fabric模组  
:::
